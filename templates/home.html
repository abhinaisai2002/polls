PollPoller

This website allows you to post some polls on different aspects by different individuals. Gearing up to the opinions of every individual, you will have a few questions in which every participant can poll according to their views. Each answer should be selected from the given set of options. Every option you choose, will be your opinion for that particular question. Extending to this, you can also comment on the imposed question when you are not satisfied with the given set of options. Hope you will find this interesting. There we go!!
